Comprobrar la conexión entre las máquinas virtuales de WSL 2: 
En una máquina virtual ponesmos "hostname -I" para ver la IP de esa distribución y en la otra ponemos "ping [dirección IP que nos salió en la primera máquina]"
Si sale una serie de datos infinitos que empiezan por 64 bytes, quiere decir que las máquinas virtuales tienen conexión entre sí
